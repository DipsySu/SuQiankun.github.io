UISearchController的使用

发表于 2016-12-29 | 分类于 iOS 开发

UISearchController 是苹果在 iOS8以后推出的行的搜索控制器.用于取代 UISearchBar;

如果我们想要使用这个控件,首先我们的拥有它;

`
@property (strong,nonatomic)UITableView *tableView;

@property (strong,nonatomic)UISearchController *searchController;

@property (strong,nonatomic)NSMutableArray *dataList;

@property (strong,nonatomic)NSMutableArray *searchList;

`

//在我们创建一个searchController,先自定义一个 继承自 UITableViewController 的QKTableViewController,作为展示搜索结果的控制器;

QKViewController *qk = [QKViewController new];
UINavigationController *nav = [[UINavigationController alloc] initWithRootViewController:qk];
//我们讲 qk 作为搜索搜索结果展示器;并且设置当前控制器为searchController 的代理

创建 tableView 后,我们需要给他设置数据源,让其展示信息:

然后我们将tableView 的 headerView 设置为 searchController 的搜索框

好了,到了这一步我们所看到应该就是一个带搜索功能的 tableView 了;

如果我们想要执行搜索,那么一定就需要获取到用户在 `searchController.searchBar`中输入的信息,那到这个信息,并且在 searchController的更新器中查询我们 tableView 中有没有查询到的数据,这里需要用到`UISearchController` 的一个代理方法

`- (void)updateSearchResultsForSearchController:(UISearchController *)searchController;`

&lt;pre&gt;`#pragma mark -- 更新器
-(void)updateSearchResultsForSearchController:(UISearchController *)searchController {
   //取出用户输入的信息,并且在 updateContentWithStr: 对比元数据源中的信息,讲符合条件的晒选出来
    NSString *searchString = [self.searchController.searchBar text];
    [self updateContentWithStr:searchString];

    //将搜索到的数据传给 qk 控制器,并在这个qk 控制器中列出信息
    if (self.searchController.searchResultsController) {
        UINavigationController *nav = (UINavigationController *) self.searchController.searchResultsController;
        QKViewController *qk = (QKViewController *)nav.topViewController;
        qk.searchList = self.searchList;
        [qk.tableView reloadData];
    }

}

-(void)updateContentWithStr:(NSString *)str{
    if (str == nil) {
        self.searchList = [self.dataList mutableCopy];
    }else{

        NSMutableArray *searArray  = [[NSMutableArray alloc] init];

        for (NSDictionary *dict in self.dataList) {

            if ([dict[@&amp;quot;sing&amp;quot;] containsString:str]) {

                NSString *str = [NSString stringWithFormat:@&amp;quot;%@&amp;quot;,dict[@&amp;quot;sing&amp;quot;]];

                [searArray addObject:str];
            }
            self.searchList = searArray;
        }
    }

}

好了,到此为止 UISearchController 的就可以成功的搜索数据啦~~~
