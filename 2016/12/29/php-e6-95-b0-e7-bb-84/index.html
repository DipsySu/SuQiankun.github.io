php 数组

发表于 2016-12-29 | 分类于 南腔北调

数组是 PHP 语言中最重要的数据类型之一,在 PHP 中的应用非常广泛;

php 是一门弱数据类型的语言,所以 php 中的数组变量可以存储任意类型,任意个数的变量,并且可以实现其他强数据类型类型中堆、栈、队列的功能；

功能异常强大，个人觉得 php 的比其他的高级语言的数组稍微那么好用那么一丢丢；

1.定义一个 php数组

  • 直接为数组元素赋值既可以声明数组

  • 使用 array() 函数声明函数

  • 多维数组的声明

    php 语言中多维数组的明明与其他语言类似,将一个一维数组放到另外一个数组中,那么这个数组就是一个二维数组

遍历数组

我们同样可以使用 for 循环对数组进行遍历

  • for 循环遍历数组

  • foreach 遍历数组

  • 使用 while + list+ each 遍历数组

使用数组指针介绍

数组指针是数组内部的组织机制,指向数组的某个元素;默认指向数组的第一个元素,通过移动或者改变数组指针的位置,可以访问数组中的任意元素;

  • current() —> 获取数组当前位置元素内容资料;
  • key() —> 获取数组当前指针所指向资料的索引值;
  • next() —> 将当前数组指针向后移动一位;
  • prev() —> 讲数组指针的为指倒回一位;
  • end() —-> 将数组指针指向最后一个元素
  • reset() —–> 将数组指针无条件的移动到第一个索引位置;
