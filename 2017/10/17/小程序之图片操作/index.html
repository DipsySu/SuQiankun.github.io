小程序之图片操作

发表于 2017-10-17 | 分类于 前端

微信为开发者提供API, 让用户能够便捷的操作手机的小相册或者拍照功能

  1. wx.chooseImage(obj)

从本地相册选择图片或者相机拍照片

  1. wx.previewImage(OBJECT)

预览图片

参数说明:

  1. current 显示当前图片的链接
  2. urls 需要预览的图片链接列表
  3. suuccess 接口调用成功回调
  4. fail 接口调用失败回调
  5. complate 调用完成回调(无论失败或者成功)
  1. wx.getImageInfo(obj)

获取图片信息

obj的参数说明

  1. src 图片路径,可以使本地文件路径 临时文件路径 网络路径
  2. success 接口调用成功回调
  3. fail 接口调用失败回调
  4. complate 接口调用回调 (无论成功失败)

success 函数中的参数说明

  1. width 图片宽度 px
  2. height 图片高度 px
  3. path 返回图片的本地路径

  1. wx.saveImageToPhotoAlbum(OBJECT)

    保存图片到相册系统, 需要获得用户授权 (scope.writePhotoAlbum)

    1. src 图片路径,可以使本地文件路径 临时文件路径 网络路径
    2. success 接口调用成功回调
    3. fail 接口调用失败回调
    4. complate 接口调用回调 (无论成功失败)

    success 函数中的参数说明

    1. errMsg 调用结果
