消息框

创建并显示对话框 , 弹出的对话框 非阻塞模式 , 用户丢氨基对话框上的按钮可以关闭或者执行某段逻辑操作 (h5 模式的对话框也可以通过closepopip 关闭) , 通过排callback函数返回用户执行的操作参数;

dialog (对话框)组件类型:

组件名称 作用
alert 警告框
confirm 确认框
prompt 输入对话框
toast 消息提示框

弹出警告信息框

  • .alert(message,title,btnVlaue,callback[,type])
  • message –> Type: String , 提示对话框上显示的内容
  • title —> String , 提示对话框上显示的标题文字
  • btnVlaue –> string , 提示对话框上按钮显示的内容
  • callback —> Function, 提示对话框关闭后的回调函数,
  • type —> value:’div’ ,是否使用h5绘制的对话框;

  • .confirm(message,title,btnVlaue,callback[,type])
  • message –> Type: String , 提示对话框上显示的内容
  • title —> String , 提示对话框上显示的标题文字
  • btnVlaue –> Array , 提示对话框上显示的内容, 有数组下标做参数
  • callback —> Function, 提示对话框关闭后的回调函数,
  • type —> value:’div’ ,是否使用h5绘制的对话框;

输入对话框

  • .prompt(message,title,btnVlaue,callback[,type])
  • message –> Type: String , 提示对话框上显示的内容
  • placeholder —> String , 提示对话框上显示的标题文字
  • btnVlaue –> string , 提示对话框上按钮显示的内容, 数组下标可做参数
  • callback —> Function, 提示对话框关闭后的回调函数,
  • type —> value:’div’ ,是否使用h5绘制的对话框;
