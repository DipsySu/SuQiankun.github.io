小程序初识

发表于 2017-10-05 | 分类于 手机端
  • 获取微信小程序的appid (不可以使用服务号或者订阅号的appid)
  • 文件结构介绍

其中app.js 是小程序的脚本文件, 可以再此文件中监听或者处理小程序的生命周期函数.生命去那句变量. 调试 读取本地数据,

app.json 是对整个小程序的全局配置。我们可以在这个文件中配置小程序是由哪些页面组成，配置小程序的窗口背景色，配置导航条样式，配置默认标题。注意该文件不可添加任何注释。

app.wxss 是整个小程序的公共样式表。我们可以在页面组件的 class 属性上直接使用 app.wxss 中声明的样式规则。

在微信给的demo中有两个页面. 一个是index 一个是log日志. 每个页面是有同一个路径下的四个不同文件构成,

  • index.js 脚本文件
  • index.wxss 样式表
  • index.wxml 页面结构
  • index.json 配置项
